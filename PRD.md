# PRD.md

## プロダクト名
安全AIサイト

## 目的
労働安全コンサルタントや現場担当者が、
法改正、気象警報、KY、保護具選定、安全対策情報を
スマホとPCで簡単に確認できるWebサイトを作る。

## MVPの目的
まずは以下の3つだけで価値を出す。

1. 法改正リスト表示
2. 各法改正のAI要約表示
3. 選択中の法改正に対する質問チャットUI

## 対象ユーザー
- 労働安全コンサルタント
- 建設・製造などの現場安全担当
- 中小企業の現場責任者
- 安全衛生に関心のある管理者

## 対応端末
- スマホ最優先
- PC対応
- レスポンシブWebサイト

## MVP機能
### 1. トップページ
- サイト名
- 一言説明
- タブナビゲーション

### 2. 法改正タブ
- 法改正カード一覧
- タイトル
- 発行日
- 概要
- 「AIで要約」ボタン
- 「この改正について質問する」ボタン

### 3. 要約表示
- 3行要約
- 現場でやること
- 対象業種
- 根拠リンク用プレースホルダ

### 4. 質問チャットUI
- 入力欄
- 送信ボタン
- ダミー回答表示
- 後でAPI接続しやすい構造

## 非MVP
以下は今回は実装しない。UIだけは将来拡張を意識してよい。
- 地域登録と警報メール通知
- 音声入力
- 画像入力
- KY生成
- 保護具選定
- 事故データベース
- ヒヤリハット蓄積
- 有料会員機能
- アフィリエイト導線

## 技術方針
- Next.js App Router
- TypeScript
- Tailwind CSS
- ダミーデータで開始
- 将来Supabase連携しやすい構造
- 将来AI API連携しやすい構造

## UI方針
- スマホでは縦並び
- PCでは2カラムまたは横並び
- 難しい操作を避ける
- 初見で何のサイトかわかることを優先

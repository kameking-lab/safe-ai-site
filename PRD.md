# PRD.md

## プロダクト名
安全AIサイト

## 目的
労働安全コンサルタントや現場担当者が、法改正・現場リスク・事故知見・朝礼要点を
スマホ/PC/サイネージで素早く確認できる「現場運用ポータル」。
安全グッズのアフィリエイトで収益化も行う。

## 対象ユーザー
- 労働安全コンサルタント
- 建設・製造などの現場安全担当
- 中小企業の現場責任者
- 安全衛生に関心のある管理者
- 朝礼広場の進行担当
- ゼネコン社内デジタルサイネージ運用担当

## 対応端末
- スマホ（縦長スクロール）
- PC（横長画面を最大限活用）
- サイネージ・常時表示モニター
- レスポンシブWebサイト

## 機能一覧
1. 法改正一覧（直近10年分、検索、AI要約、質問チャット）
2. 今日の現場リスク（地域選択・作業種別）
3. 事故データベース（既存 + 厚労省 死亡/休業災害DB 10年分モック）
4. /signage サイネージ画面（日本地図ハッチング + 報道労災 + 法改正要約）
5. Eラーニング（テーマ一覧・クイズ）
6. KY用紙（紙様式準拠の危険予知活動表、音声入力対応）
7. PDF出力（KY・朝礼要点）
8. 通知/メール配信設定
9. 音声入力（全テキスト入力欄対応）
10. 安全グッズ（分野別アフィリエイト）
11. サイドバーナビゲーション（ルート遷移）

## 技術方針
- Next.js App Router
- TypeScript
- Tailwind CSS
- モックデータで段階開発（外部API本接続・DB本接続は不可）
- 将来Supabase連携しやすい構造
- 将来AI API連携しやすい構造
- Vercel でデプロイ・公開

## UI方針
- スマホでは縦並びスクロール
- PCでは max-w-7xl + mx-auto でコンテンツ幅を制限し、横に間延びさせない
- 初見で何のサイトかわかる
- 朝礼と常時表示で1〜2秒で意味が伝わる短文優先

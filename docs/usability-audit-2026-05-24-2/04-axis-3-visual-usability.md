## 視線誘導の問題

### V1: /chatbot で視線が17色17ブロック間で迷子
- **要素列**: 緑(RAG📚) / 橙(AI🤖) / 緑(信頼度🟢) / 黄(🟡) / 赤(🔴) / エメラルド(📎出典) / スカイ(📚関連) / バイオレット(🔍深く) / アンバー(通達) / スカイ(リーフレット) / 青(フォローアップ) / ロゼ(レポート) / バイオレット(計画) / エメラルド(KY) / バイオレット(化学) / スカイ(法改正) / アンバー(サイネージ) = **17色以上が同一回答内で混在**
- **影響**: 色による情報設計が崩壊。職長は「赤=危険」「緑=OK」等の身体感覚を持っているが、ここでは色が意味を失う。
- **修正**: 配色ガイドライン再設計。主要3色(出典=エメラルド・関連=スカイ・警告=ロゼ)に統合。
- **工数**: 8h

### V2: /accidents-reports/[industry] 14セクション の優先順位なし
- **存在するセクション**: サマリ/危険要因/重大事故/事故型Top10/原因Top10/業種特有パターン/時間帯規模/季節性/年次/推奨対策/30項目チェック/関連法令/比較CTA/次のアクション = **14ブロック**
- **影響**: スマホで「最初に何を読むべきか」不明。職長が朝礼前に開いた瞬間に離脱。
- **修正**: タブ化(サマリ/事故詳細/対策チェック/法令)+「優先順位順」のチップで上位3項目をハイライト。
- **工数**: 12h

### V3: ホーム3層重複(FlagshipGrid + ThreePillars + 建設業バナー)
- **状況**: ホームに「FlagshipGrid 10カード」+「HomeThreePillars(死亡事故/警報/法改正)」+「建設業の方はこちら」+その他CTAが縦積み。
- **影響**: ファーストビューを `HomeThreePillars` が食う+下に行くまで建設業バナー見えない。
- **修正**: ヒーロー直下に建設業バナー→ HomeThreePillars→ FlagshipGrid の順。
- **工数**: 4h

### V4: ヒーローH1が抽象的
- **状況**: H1「労働安全衛生のAI・DX活用ポータル」+ H2「現場の安全を、AIで変える。」(`new-home-hero.tsx:62-68`)
- **影響**: 抽象的・何ができるか分からない。
- **修正**: H1を動詞・名詞列挙に置換「KY・事故DB・安衛法AIチャットを1画面で」。
- **工数**: 2h

## 色・タイポグラフィの統一感

### 良い点(維持)
- グローバル: emerald-600(プライマリ)+slate(本文) が一貫
- CardGrid 配色: amber/blue/emerald/rose/violet で機能を識別
- /law-hierarchy のカラーバッジ(法→政令→省令→告示) が階層意味と一致

### 問題点
- /chatbot で17色混在(前述)
- /signage で絵文字依存(🌅☕🌆📱🖥)が現場サイネージとして安っぽい
- /mental-health-management のリスト型表示が他ハブと不統一(他は3カードグリッド)

---

## モバイル下部ナビ(MobileBottomNav)の評価

### 良い点(Wow水準)
- 5タブ(ホーム/KY/AIチャット/日誌/もっと) が職長日常を捕捉
- 48px以上タップ領域、safe-area-inset-bottom 対応
- ESC で「もっと」シート閉じる、body スクロール停止 → UX良
- 「もっと」シートに化学物質RA/法令検索/事故分析/年次計画/気象リスクの5項目 → 主要機能網羅

### 改善余地
- 「もっと」シートが3列グリッドで5項目=6スロットで1つ空(`MobileBottomNav.tsx:154-160`) → 「条文番号入力」「サイネージ起動」を追加すれば6項目で埋まる
- 主要4タブのアイコンが Home/ClipboardList/MessageSquare/FileText で似てる(灰色基調) → アクティブ色(emerald)で区別はOK

## グローバルナビ NAV_CATEGORIES の評価

### 問題: 10カテゴリ40+項目
- 「マップ」「学習」「法律」「現場ツール」「事例・データ」「多様な働き方」「ツール」「プロジェクト」「その他」+ホーム = **10カテゴリ**
- 「マップ」=サイネージ+気象リスク(意味希薄)
- 「学習」=Eラーニング1項目のみ
- 「その他」=辞書/FAQ/グッズ/助成金/通知 = カテゴリ機能を果たしていない雑然
- ハンバーガーで縦スクロール地獄

### 修正案: 6カテゴリ20項目以内
1. **質問する** (チャット/法令検索/法令体系/通達/用語)
2. **記録する** (KY/日誌/PDF)
3. **分析する** (事故DB/事故レポート/化学物質)
4. **学ぶ** (Eラーニング/法定教育/FAQ/事例)
5. **業種別** (建設/製造/医療福祉/運輸/サービス)
6. **管理** (年次計画/熱中症/メンタル/サイネージ)

工数: 8h

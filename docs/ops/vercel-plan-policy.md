# Vercel プラン運用方針

最終更新: 2026-05-20
決定者: オーナー（Dispatch経由確認）

---

## 現在の方針: Pro プラン継続（無期限）

安全AIポータル（anzen-ai-portal.jp）は **Vercel Pro プランを無期限継続** する。

---

## 決定経緯

### 2026-05-19 時点の状況

Dispatch A（CDNキャッシュ設定 / PR #253）および Dispatch B（ISR Writes削減 / PR #252）を実施した後も、以下のクォータがHobby上限を恒常的に超過することが確認された。

- **ISR Writes**: Hobby上限 200,000/月 に対し推定 1,100,000/月（約550%）
- **Edge Requests**: Hobby上限 1,000,000/月 に対し推定 1,600,000/月（約160%）

Hobby復帰判定モデル（14日trailing avg × 30日）による復帰可能性は **5%未満** と算定。

### 構造的制約

- サイネージ（`/signage`）・ISR静的再生成の仕組み上、ISR Writes を Hobby 上限内に収めることは機能削除を伴わない限り不可能
- Edge Requests は AIクローラーブロック + CDNキャッシュ最適化後でも Hobby 上限の1.6倍
- これ以上の削減は機能品質の大幅な劣化を招く

---

## Pro継続の理由

1. **技術的理由**: 上記のクォータ超過により Hobby への復帰が構造的に不可能
2. **運用コスト**: Pro プランの月額コストはポートフォリオサイトとしての受注機会損失より十分に小さい
3. **サービス安定**: Pro でのサービス停止リスクがないため、協会向けデモ・受注活動に支障なし

---

## 再評価条件

以下のいずれかが発生した場合のみ、Hobby復帰または他プランへの移行を再評価する。

1. Vercel の月額料金が現在の2倍以上に値上がりした場合
2. Vercel がプラン体系を大規模改定し Hobby のクォータが実用的に拡大した場合
3. サイトのアクセス量が著しく減少し ISR Writes が 150,000/月 を下回った場合
4. オーナーが明示的に再評価を指示した場合

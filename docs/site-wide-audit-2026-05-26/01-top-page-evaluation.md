# 01. トップページ評価（軸1）

評価: **84 / 100**

## トップの構成（上から）
1. `NewHomeHero` — タグライン「労働安全衛生のAI・DX活用ポータル／現場の安全を、AIで変える。」＋**3大CTA（AIチャット・業種別事故分析レポート・年次安全衛生計画）**＋KY/法令検索のショートカット＋統計3枚（通達数・事故10年・保護具DB、各出典リンク付き）
2. 「建設業の方はこちら」帯（`/for/construction`、P0-017で昇格）
3. `HomeThreePillars`「本日の安全トピック」（死亡事故/気象警報/法改正＋AI注意喚起文生成）
4. `FlagshipGrid`「主要機能」**10枚**（打合せ書・KY・化学物質RA・サイネージ・法改正・AIチャット・事故・教育・業種・作業環境測定）

## 良い点
- **3秒で「何のサイトか」が伝わる**: H1＋サブ＋一文説明で「労働安全のAI/DXポータル・建設/製造/運輸/医療福祉/林業」が明確。
- **「無料」明示**: metadata と説明文に「無料」。フッターに「個人運営の研究プロジェクト」。
- **CTAが具体的動詞**: 「労働安全衛生法をAIに質問」等。FlagshipGrid も `FEATURE_CTA_JA` で「KY用紙を作る」等の動詞ラベル。
- **トップが“今日使える”**: 本日の死亡事故・警報・法改正が動的表示。出典リンク付き。
- **統計に出典**: 各KPIに `SITE_STATS_META` の出典URL／asOf。誇張なし。

## 問題点
- **【P0】“主要機能”の定義がトップ内で割れている**: ヒーローの3大CTA＝AIチャット・**事故分析レポート**・**年次計画**。しかしその下の FlagshipGrid 10枚に「事故分析レポート」「年次計画」は**独立カードとして無い**（前者は事故カードのサブ、後者は法改正カードのサブ）。「結局、推されているのはどれ？」と初訪問者が迷う。フッター「主要機能」も同3つで、グリッド10と不一致。
- **【P2】ロングテールがトップに出ない**: 熱中症・メンタル・石綿・外国人・助成金・多様性はトップに導線が無く、FlagshipGrid 10枚＋「機能一覧（全機能）→」リンクからのみ。約150ページの広さが初見では伝わらない。
- **【P2】「初めての方へ」が無い**: オンボーディング（quick-tour）への導線がトップに無い。
- **【P3】信頼アンカーが薄い**: ヒーロー上部に「労働安全コンサルタント監修（登録260022）」の一言が無く、権威の即時提示ができていない（→ `07-trust-authority.md`）。

## 改修方針
- 主要機能の定義を**1つに統一**（ヒーロー3CTAを“最初の一歩”、FlagshipGrid を“全機能の代表”と役割分離して言語化する／または3CTAをグリッド先頭にも置く）。詳細は `02` `13`。
- ヒーローに「労働安全コンサルタント監修」バッジ1行（軽微・低リスク）。

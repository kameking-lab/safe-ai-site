# 09 多言語対応（軸9）

## 現状
- サイトに `contexts/language-context` + `useLanguage` + `TranslatedPageHeader` の i18n 基盤あり（ページ見出し等の翻訳）。
- 化学物質の**規制解説・SDS要点・教育資料の多言語化は未対応**。

## 設計案（既存資産流用）
- M-1 固定ラベル多言語: 化学物質画面の固定ラベル（規制名・見出し・「対策」等）を language-context の辞書に追加。サイネージで確立した6言語パターン（日/英/越/中/タガログ/インドネシア）を流用。
- M-2 規制解説のAI翻訳併記: regulation-tag-labels の解説文を Gemini でオンデマンド翻訳併記（参考訳明記）。技能実習生・特定技能者が「この物質はなぜ危険か」を母国語で把握。
- M-3 SDS要点の多言語: D-1 SDS取込みと連動し、要点を母国語で表示。

## コスト/リスク
- 固定ラベルは辞書化（API不要）。AI翻訳は回数制限＋キャッシュで¥10,000閾値内。
- 安全情報のため「参考訳・AI生成」明記必須。

## 優先度
- P1: M-1 固定ラベル多言語（API不要・低リスク）。
- P2: M-2 規制解説のAI翻訳併記、M-3 SDS要点多言語。

外国人労働者への母国語配慮は安衛法の安全衛生教育の趣旨に合致（軸2/法令面）。

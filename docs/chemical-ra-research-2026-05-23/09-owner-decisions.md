# 09: オーナー判断必須事項 一覧

作成日: 2026-05-23
件数: 9件
優先度: ★ = 低、★★★★ = 高

各項目には「論点」「選択肢」「推奨」「必要時期」「未決時の影響」を記載。

---

## 項目1: NITE-CHRIP データの利用範囲 ★★★

### 論点
NITE 化学物質総合情報提供システム（CHRIP）の取込範囲。GHS分類結果のみか、SDS本文も含むか。

### 選択肢
- A: GHS分類結果のみ（約3,300物質、安全）
- B: GHS分類結果 + 規制情報リンク（推奨）
- C: SDS本文の自動取込み含む（要法務確認）

### 推奨
**B**: GHS分類結果＋規制情報リンク。SDS本文取込みは Phase 5 で別途検討。

### 必要時期
Phase 1 着手前

### 未決時の影響
Phase 1 のスコープ確定不可。物質数目標が確定しない。

---

## 項目2: 日本産業衛生学会（JSOH）許容濃度の引用範囲 ★★★★

### 論点
現状 `concentration-limits.json` に 91件の JSOH 数値が格納されている。学会著作物の数値転載のリスク。

### 選択肢
- A: **全件削除**、出典明示のリンクのみ表示（最安全、推奨）
- B: 数値転載を当面維持し、学会に許諾照会
- C: 数値を区分表記に丸める（10-50ppm レンジ表記）

### 推奨
**A**: Phase 1 の最初のタスクで全件削除＋リンク化。

### 必要時期
Phase 1 着手前

### 未決時の影響
著作権リスク継続。商用化（サブスク開始）時の法的脆弱性。

---

## 項目3: ACGIH TLV 数値の引用範囲 ★★★★

### 論点
現状 ACGIH TLV 502件が `concentration-limits.json` に格納。商用利用の制約が厳しい。

### 選択肢
- A: 全件削除、出典明示のリンクのみ（最安全、強く推奨）
- B: 当面維持しACGIH に許諾照会
- C: 区分表記に丸める

### 推奨
**A**: 項目2と同じ理由。Phase 1 で全件削除。

### 必要時期
Phase 1 着手前

### 未決時の影響
項目2と同じ。商用化時の法的脆弱性。

## 項目4: メーカーSDS の自動取込み（Phase 5 全体の可否） ★★★

### 論点
Phase 5 で計画する SDS PDF の自動収集・本文抽出は、著作権リスクが非常に大きい。

### 選択肢
- A: **Phase 5 を実装しない**（最安全、推奨）
- B: NITE-CHRIP の公開SDS連携のみ（限定的）
- C: 完全実装（要法務、要許諾）

### 推奨
**A**: Phase 1-4 完了で十分な差別化が可能。Phase 5 は将来検討に留める。

### 必要時期
Phase 5 着手前（Phase 4 完了時点で再検討）

### 未決時の影響
Phase 5 のスコープ・予算確定不可。ただし Phase 1-4 着手には影響なし。

## 項目5: PDF出力ライブラリの選定 ★★

### 論点
1枚紙RA の PDF出力をどの技術で実装するか。

### 選択肢
- A: ブラウザ印刷ダイアログ + HTML/CSS（追加依存ゼロ、推奨）
- B: `@react-pdf/renderer` (サーバ生成、UI再実装)
- C: `jspdf` + `html2canvas` (クライアント生成、画質劣化)

### 推奨
**A**: Phase 2 で最速実装。`@react-pdf/renderer` への移行は Phase 4 以降に再検討。

### 必要時期
Phase 2 着手前

### 未決時の影響
Phase 2 の実装方針確定不可。

## 項目6: 業種テンプレートの初期スコープ ★★

### 論点
詳細判定モードに搭載する業種テンプレートを4業種で確定するか、絞り込み or 拡張するか。

### 選択肢
- A: 4業種（塗装・防水・解体・地盤改良、推奨）
- B: 2業種（塗装・解体のみ、Phase 3短縮）
- C: 6業種以上（製造業・清掃業 等を追加）

### 推奨
**A**: 建設業の主要4業種で十分カバー。製造業等は Phase 4 以降に追加。

### 必要時期
Phase 3 着手前

### 未決時の影響
Phase 3 の実装スコープ・工数確定不可。

## 項目7: 複合曝露アルゴリズムの選択 ★★

### 論点
混合物・複数物質併用時の補正方式。

### 選択肢
- A: ACGIH 推奨の加算法 Σ(Ci/TLVi) ≤ 1（標準、推奨）
- B: 厚労省式の TLV 相互作用（複雑、現場運用が困難）
- C: 加算法 + 相乗作用警告ペア（既知の組合せのみ警告、A+α）

### 推奨
**C**: 標準加算法に加え、ベンゼン+トルエン等の既知相乗作用ペアを警告表示。

### 必要時期
Phase 3 着手前

### 未決時の影響
Phase 3 の判定アルゴリズム確定不可。

## 項目8: 免責表示の文言確定 ★★★★

### 論点
誤判定時の責任範囲を明確化する免責表示。

### 推奨案
> 本ツールは化学物質リスクアセスメントの支援を目的とした参考情報を提供します。
> 最終的な判断は事業者・労働安全衛生コンサルタント・産業医・専門家が行ってください。
> 本ツールの判定結果に基づく労働災害・健康被害について、運営者は一切の責任を負いません。
> 関連法令の遵守は事業者の責務です。

### 検討事項
- 上記文言で良いか、または法律事務所監修の文言に差し替えるか
- 表示位置（1枚紙RAフッター・全画面・PDF・利用規約）
- ユーザー登録時の同意取得方式

### 推奨
**1枚紙RAのフッター・PDF出力・利用規約の3箇所に表示**。ユーザー登録時にチェックボックス同意。

### 必要時期
Phase 2 着手前（1枚紙RAに表示する文言が必要）

### 未決時の影響
Phase 2 の UI 確定不可。法的リスク残存。

---

## 項目9: 「監修」表記の有無 ★★

### 論点
社長（労働安全コンサルタント）の「監修」表記を入れるか。

### 選択肢
- A: 監修表記なし（運営: 「安全AIポータル運営チーム」とする、最安全）
- B: 「監修: 労働安全コンサルタント ●●●●」表記
- C: 「監修・運営: 労働安全コンサルタント ●●●●」（個人運営研究プロジェクト体裁強化）

### 論点
- 監修表記は信頼性を高めるが、監修者の責任が発生
- 個人運営研究プロジェクトとして体裁を整える場合は C が整合的
- 課金開始後は監修者の損害賠償リスクが上がる

### 推奨
**B**: 信頼性を訴求しつつ、利用規約と免責で監修者の責任範囲を限定する。

### 必要時期
Phase 2 着手前（1枚紙RAに監修表記を入れるか確定）

### 未決時の影響
Phase 2 の UI・利用規約確定不可。

## 着手順序の推奨

1. **項目2, 3 (JSOH/ACGIH)** を Phase 1 開始の直前に決定 → Phase 1 のスコープ確定
2. **項目1 (NITE-CHRIP)** を Phase 1 開始時に決定 → ETL 開始
3. **項目5, 6, 8, 9** を Phase 2 開始前に決定 → 1枚紙UI と免責文言を確定
4. **項目7** を Phase 3 開始前に決定 → 詳細判定アルゴリズム確定
5. **項目4** を Phase 4 完了時に再検討 → Phase 5 実装可否確定

オーナーが全項目に推奨選択肢で確定すれば、本ロードマップ全体（Phase 1-4 計 16-24日）が即時着手可能。

# 04: データソース棚卸し + 著作権整理

作成日: 2026-05-23
範囲: 化学物質RA万能化に使用候補のデータソースを、取得手段・著作権・更新頻度・信頼性の4軸で評価。

## 3. データ取得手段の総括

```
[公的データソース]
   ├─ MHLW告示・通達（PDF/Excel）
   │   → scripts/etl/ 配下に取込スクリプト追加
   ├─ NITE-CHRIP（CSV/Web）
   │   → fetch-nite-chrip.mjs を新規作成
   ├─ 環境省/経産省（PRTR、化審法）
   │   → 別表のExcel/CSV取込
   └─ IARC Monographs
       → CSV/手動アノテーション

[民間データ]
   ├─ JSOH 学会誌
   │   → 出典明示・リンクのみで安全運用
   ├─ ACGIH TLV
   │   → 出典明示・リンクのみで安全運用（数値非掲載検討）
   └─ メーカー SDS
       → URL リンクのみ、本文取込みは要法務

[業種特化]
   └─ 建災防・国交省指針
       → 出典明示で再利用、PDF直リンク
```

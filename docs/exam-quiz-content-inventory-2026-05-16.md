## カテゴリ A: カリキュラム網羅型 完全創作

**ディレクトリ:** `web/src/data/mock/quiz/cert-quiz/`  
**UI 接続:** `/exam-quiz` (メイントップ + `/exam-quiz/[slug]` プレーヤー)  
**問題数:** 10 資格 × 100 問 = **1,000 問**

### 内容説明

`types.ts` に明示: 「既存の `web/src/data/exam-questions/` は過去問ベース、こちらはカリキュラム網羅型の curated 100問セット。」

各ファイルは出題範囲シラバスから独自に設計した 4 択問題。法令根拠・解説付き。年度付き ID なし。逐語転載の証拠なし。

|---|---|---|---|

| `anzen-civil.ts` | 労働安全コンサルタント（土木） | 100 | 基礎〜応用 |

| `health-1st.ts` | 第一種衛生管理者 | 100 | 基礎〜標準 |
| `health-2nd.ts` | 第二種衛生管理者 | 100 | 基礎〜標準 |

### 法的リスク評価

| 観点 | 評価 |
|---|---|
| 逐語転載リスク | **なし** — 設計がカリキュラム網羅型であり独自表現 |
| 景品表示法（優良誤認） | **低** — PR #188 で UI ラベルを「学習用クイズ」に修正済み |

| 試験実施機関との競合 | **低** — 「過去問」と銘打たない限り問題なし |

### 推奨アクション

**売り込み可** — メイン機能 `/exam-quiz` の中核コンテンツ。「出題形式を踏まえた学習用問題」として積極展開可。

## カテゴリ C: 年度付き・ソース注記なし（安全コンサルタント本試験 年度別）

**ディレクトリ:** `web/src/data/exam-questions/`  
**UI 接続:** `/exam-quiz` 内の `ExamQuizClient`（年度フィルタで表示）  
**問題数:** 約 **300 問**

### ファイル別内訳

| ファイル | 対象試験・年度 | 問題数 | ソースコメント |
|---|---|---|---|
| `2023-safety-general.ts` | 労働安全コンサルタント 産業安全一般 2023 | ~60 | なし |
| `2023-safety-law.ts` | 労働安全コンサルタント 産業安全関係法令 2023 | ~60 | なし |

| `2025-safety-general.ts` | 労働安全コンサルタント 産業安全一般 2025 | ~30 | なし |
| `2025-safety-law.ts` | 労働安全コンサルタント 産業安全関係法令 2025 | ~30 | なし |

### 問題ファイルの構造的特徴

- ファイル名・ID に実際の試験年度（2023/2024/2025）を含む
- 問題 ID 形式: `"2023-sg-001"` など、試験年度が ID に埋め込まれている
- ソースコメントなし — 創作か転載かをコード上で判別できない
- 安全コンサルタント試験は中央労働災害防止協会（JISHA）/安全衛生技術試験協会（JAISH）が実施

### 法的リスク評価

| 観点 | 評価 |
|---|---|
| 逐語転載リスク | **不明（コード上の判断不可）** — ファイル名と ID が実年度と一致するため、実際の試験問題を転写した可能性を排除できない |
| 景品表示法（優良誤認） | **高** — 「2024年度試験の問題」として解釈される UI 表示は優良誤認に当たりうる |
| 著作権 | **不明〜高** — 試験問題は著作物性を有し、JISHA/JAISH が著作権を保有する可能性がある |
| 試験実施機関との競合 | **高** — 実年度を冠した問題セットは実施機関の公式過去問と直接競合するように見える |

### 推奨アクション

**削除候補（要オーナー判断）** — 作成時のソースが不明なため、逐語転載の有無を確認できない。確認できるまでは `ExamQuizClient` の年度フィルタから非表示化、またはファイルを削除することを推奨。オーナーが問題の出典を確認し、完全創作であることが証明できた場合は免責コメントを追加の上 維持可。

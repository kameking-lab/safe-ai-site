# 採用レビュー（明日から使うか / 使わないか）— エグゼクティブサマリ

作成: 2026-05-31 / Opus 4.8 / 評価軸はただ一つ「現場のプロが明日から乗り換えて使うか」

> 前提（厳しく適用）: 現場は今、紙・Excel・公式サイトで“回っている”。乗り換えには覚えるコスト・手順変更コスト・元請や同僚への説明コストがかかる。これを上回る“革命的便益”が無ければ人は動かない。「無料・登録不要」「AI付き」は入口を下げるだけで、乗り換え理由ではない。

## 9機能の判定一覧

- KY用紙 (/ky/paper) ……… **あと一歩**（毎朝の手順を変える壁＋元請様式の縛り。連携・保管は価値だが決定打に一歩足りない）
- 安全工程打合せ書 (/safety-diary) … **あと一歩**（痛みは大きいが、集約の手間を元請が全部背負う構造のまま）
- サイネージ (/ky/morning) ……… **使わない**（大画面モニタという設備が前提。中小現場の多くに無い）
- 化学物質RA (/chemical-ra) ……… **使う（9機能で最有力）**（2024自律管理の法的義務×締切×時短×記録様式。動機が構造的に強い）
- 事故DB (/accidents) ……… **使わない**（データ源は厚労省と同じで収録292件。公式あんぜんサイトの網羅性に劣り「これでなければ」が無い）
- チャットボット (/chatbot) ……… **あと一歩（入りやすさは最良）**（スイッチングコスト最小。ChatGPTより安全だが「これでなければ」は弱い）
- 法改正 (/laws) ……… **あと一歩**（メール購読というプッシュは実装済み。ただし“向こうから届く”確実性と業種特化が鍵）
- 新着情報 (/whats-new) ……… **使わない（単独では）**（そもそも乗り換え対象が無い＝新習慣の創出が必要。玄人向けRSS）
- 労災裁判例 (/court-cases) ……… **使わない（“調べる”用途では）**（16件は網羅性ゼロ。“説明する/学ぶ”用途なら価値）

## 最も乗り換え動機が強い / 弱い

- 最強: **化学物質RA**。理由は機能の出来ではなく“外圧”——2024年の自律的管理で多くの事業者が「今までやっていなかったRAを、新たに・期限内に・記録付きで」やらねばならない。厚労省CREATE-SIMPLE＋SDS手読みより明確に速く、A4記録まで出る。これは「乗り換え」ではなく「ゼロからの採用」なのでスイッチングコストの壁が低い。
- 入りやすさ最強: **チャットボット**。チャットは学習コストほぼゼロ。ただし“使い続ける決定打”は弱い。
- 最弱: **新着情報**（乗り換え対象が存在しない）／**労災裁判例**（件数不足で実務の「調べる」に使えない）／**サイネージ**（設備前提）。

## このサービス全体は現場のやり方を変えさせられるか — 総括

**正直な結論: 機能単位では、ほとんどが「単独で既存手段を置き換える」決定打を欠く。** 「機能が揃い・品質が高い」ことは確認できたが、それは「採用される」とは別物だ。現場は機能ごとに乗り換えを判断し、現状維持バイアスは強い。

本サービスの本当の強みは個々の機能ではなく、「労働安全衛生の実務が1か所に集約され、KY→サイネージ、物質→RA→記録、事故→法令→判例、が相互に連携する」**ポータルとしての統合価値**にある。だがこの統合価値は“使い始めた後”にしか効かない。初見の現場プロには見えない。

したがって現実的な採用戦略は「全機能を均等に磨く」ではなく、**乗り換え動機が構造的に強い入口（=化学物質RA：法的義務、=チャットボット：低コスト）に集中投資し、そこから他機能へ回遊させる**こと。横並びで“あと一歩”を量産しても、現場は動かない。

各機能の詳細評価は 01〜09、優先是正（乗り換えを起こす一押し）は 10-verdict.md に記す。

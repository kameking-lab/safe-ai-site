# 03 競合SaaS・建設業ツール比較（軸3）

調査日 2026-05-26。**実在確認できたもののみ**記載。確認できないものは除外/明記。具体数値は出典で確認できたもののみ。

## 実在確認できた競合

1. **クェスタ「ご安全にモニター」** — 建設現場専用のデジタル朝礼看板（タッチ式大型モニター）。事故事例配信・朝礼コンテンツ・図面・施工体系図。タブレット/PCから遠隔更新。**当社と最も用途が近いが専用ハード前提**。価格例: 75型 200万円/86型 298万円/100型 460万円（税別）、仮囲い電子看板86〜94万円。無料枠は確認できず。
   - https://questar.ac/
2. **らくらく現場 Webサイネージ（電子朝礼看板）** — 国産クラウド型。施工体制台帳/安全管理/工程/工事黒板など12種のサイネージ機能。スマホ・タブレット表示。**機能的に最も近い**。料金・無料枠・端末共有は出典に記載なし（要問合せ）。
   - https://rakuraku-genba.jp/service/06-tablet-constructionsite-signage.html
3. **Buildee 労務安全/調整会議/入退場** — 安全書類(グリーンファイル)・作業間連絡調整・CCUS入退場（顔認証/検温）。協力会社は無料、元請有料。**朝礼の全画面サイネージ表示は非対応**。
   - https://service.buildee.jp/
4. **ANDPAD** — 建設DX総合PF（黒板/写真/工程/チャット）。月額ID課金。朝礼全画面サイネージ専用機能は確認できず。
   - https://andpad.jp/products
5. **SPIDERPLUS** — 図面起点の施工管理（電子黒板・検査・帳票）。1ID 月3,300円＋初期44,000円、2か月無料トライアル。サイネージ機能は確認できず。
   - https://kensetsu.gemba-tech.jp/reviews/spiderplus/
6. **現場ポケット** — 掲示板/トーク/日報。アカウント無制限。月11,000円(税込)/年契約9,680円、最大2か月無料。サイネージ専用機能は確認できず。
   - https://gempo.info/plan
7. **汎用サイネージSaaS（Yodeck/NoviSign/ScreenCloud）** — Yodeckは1画面無料、有料$8〜/画面・月。建設・KY特化テンプレや日本語現場運用は確認できず。
   - https://www.yodeck.com/pricing/ ／ https://www.novisign.com/pricing/ ／ https://screencloud.com/pricing

### 確認できなかったもの（除外）
- 「Conoa」: 建設朝礼サイネージ製品としての実在を**確認できず**（除外）。
- ダンドリワーク: 製品は実在するが、サイネージ/朝礼掲示機能の言及を確認できず（競合一覧から除外）。

## 当社が刺さる差別化（5件）
1. ハード不要・初期費ゼロ（クェスタの大型モニター200万円〜が要らない）。
2. 完全無料＋登録不要で即利用（現場ポケット月11,000円/SPIDERPLUS 初期費＋ID課金/ANDPAD ID課金と対照）。
3. 6桁コードの端末間共有が軽量・独自（らくらく現場/クェスタで確認できず）。当日参加の協力会社にもURLログイン不要で共有。
4. KY朝礼の全画面表示に特化（Buildee/ANDPAD/SPIDERPLUS/現場ポケットは朝礼全画面サイネージ専用機能が確認できず）。
5. 「建設安全特化×無料×日本語」の同時成立（汎用SaaSは安価でも建設・KY向けでない）。

## 競合にあり当社に無い、現場に有用な機能（5件）
1. CCUS連動の入退場管理（顔認証/検温）— Buildee。※当社方針（現場に何かさせない・認証なし）と相反するため要慎重。
2. 安全書類の期限自動チェック — Buildee。
3. 電子小黒板付き写真・改ざん検知 — ANDPAD/SPIDERPLUS。
4. 工程表/図面/施工体系図の表示連携 — 各社・らくらく現場。朝礼画面に当日工程を出せる。
5. 本部から複数現場・複数画面への一斉配信 — クェスタ/汎用SaaS。

## 結論
直接競合で最も近いのは**クェスタ（高額ハード型）**と**らくらく現場（料金不透明）**。
当社の立ち位置は明確: 「無料・登録不要・手持ち端末だけ・6桁共有」で**導入障壁が競合中最小**。
中小ゼネコン/一人親方という未開拓層に刺さる。機能面で全部追う必要はなく、朝礼掲示体験の完成度で勝負。

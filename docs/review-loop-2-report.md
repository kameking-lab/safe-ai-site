## 実施した修正

### 1. 教育詳細 5 ページの「関連条文」を e-Gov 直リンク化

| ファイル | 旧 | 新（追加した直リンク） |
|---|---|---|
| `roudoueisei/necchu/page.tsx` | 平文「労働安全衛生規則／労働安全衛生法 第22条」 | 安衛則 第612条の2（2025年6月施行）／安衛法 第22条 |
| `roudoueisei/shindou/page.tsx` | 平文「労働安全衛生規則 第13条」 | 安衛則 第13条 |
| `roudoueisei/souon/page.tsx` | 平文「労働安全衛生規則 第13条」 | 安衛則 第13条 |
| `roudoueisei/youtsu-yobou/page.tsx` | 平文「労働安全衛生法第59条、第60条の2」 | 安衛法 第59条／同 第60条の2 |
| `hoteikyoiku/chemical-ra/page.tsx` | 平文「労働安全衛生規則 第34条の2の7／第577条の2」 | 安衛法 第57条の3／安衛則 第34条の2の7／同 第577条の2 |

リンク先は全て `laws.e-gov.go.jp/law/347AC0000000057`（労働安全衛生法）または
`laws.e-gov.go.jp/law/347M50002000032`（労働安全衛生規則）の一次情報源。
これで全 12 教育詳細ページが「条文 → e-Gov」の一貫した動線を持つ。

### 2. 景表法 NG 表現の調査

`確実に / 完全に / 100% / 業界No.1 / 日本一 / 最大級 / 当社調べ / 必ず[達成・効果]` を
`web/src/app` 全体で検索したが、ユーザー向け文言には**該当なし**。

検出された 100% は CSS グラデーション値、必ずは API ルートのプロンプト（chatbot 自身への指示）で、
表示文言ではないため除外。

### 3. tokubetsu 6 ページとの構造統一

tokubetsu 系 6 ページ（ashiba/fullharness/kensaku-toishi/sankesu/tamakake/teiatsu-denki）は既に
`<a href="https://laws.e-gov.go.jp/law/347M50002000032">` パターンを採用していたため、
roudoueisei + chemical-ra に同じ構造を適用することで全 12 ページが統一された。

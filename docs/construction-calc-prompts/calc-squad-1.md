# 建設計算 部隊1 — 玉掛用具の安全率系（チェーン・繊維スリング・シャックル・つり上げ装置定格）

あなたは建設計算コーナーの量産部隊です。共通スキーマ（`web/src/lib/construction-calc/schema.ts`
の `ConstructionCalculator`）に準拠して、以下の計算機を **1機=1PR** で上から順に実装してください。
基盤（計算書PDF・AI解説窓・AI入口）は自動で付きます。**計算は決定論の純関数**、**出典未確認の
数値は載せない（＝入力値にする）**、**安全側に丸める**、**強警告を持つ**。完了条件・ゲート・
自己 squash マージ承認は `README.md` を参照。

既存の玉掛けワイヤ（`sling-wire-load`）と相互リンク（basis 説明文や cautions で言及）すること。

## 1-A. つりチェーンの安全荷重（`chain-sling-load`）★P0

**用途**: つりチェーン（チェーンスリング）の安全使用荷重を、掛け方・吊り角度・安全係数で判定。

**根拠・数式（一次資料で確認）**:
- クレーン等安全規則 第213条の2（つりチェーンの安全係数）:
  - 切断荷重 ÷ 安全係数 が使用荷重以上であること。
  - 安全係数は原則 **5以上**。ただし次の①②両方を満たすものは **4以上** で可:
    ① 伸びが製造時の長さの0.5%を超えないこと ② 断面直径の減少が製造時の10%を超えないこと。
  - → **安全係数は select（4／5）＋条件チェックの注記**で持つ。条文の条件を warnings に明記。
- 掛け数・吊り角度による張力増加は `sling-wire-load` と同じモード係数方式（2点吊り・角度係数
  1/cos(θ/2)、または JIS B 8817 等の吊り角度係数）。**係数の出典は sling-wire-load の basis を踏襲**。
- チェーンの等級別（V/VLレベル、8級/10級/100等）切断荷重は **JIS B 8817（チェーンスリング）**
  または製造者証明書の値。**等級別・径別の切断荷重表は一次資料で径ズレに注意して確認**。
  確認できない場合は「切断荷重を入力」方式（メーカー証明書の値）にする。

**入力（fields）**:
- `loadKg`（number, kg, 100〜100000）荷の質量
- `legs`（select）1本吊り／2点吊り／3点4点吊り（4本は3本扱いの安全側・sling-wire-load 踏襲）
- `angle`（number, °, 0〜120）つり角度（2点以上のとき）
- `breakingLoadKn`（number, kN）チェーン1本の切断荷重（JIS表 or 証明書。**既定は控えめ or 0で要入力**）
- `safety`（select）安全係数 5（原則）／4（伸び0.5%以下かつ径減少10%以下の条件を満たす場合）

**外部突合テスト**: JIS B 8817 の吊り角度係数・モード係数表、または日本クレーン協会テキストの
計算例と一致（角度0°/60°/90°/120° の境界、安全係数4と5の切替）。
**強警告**: 安全係数4を選べるのは条文の①②を満たすときだけ・点検（変形/伸び/摩耗）・
リンク損傷時の使用禁止・作業主任者/玉掛技能講習。

## 1-B. 繊維スリング（ベルトスリング）の安全荷重（`fiber-sling-load`）★P0

**用途**: ベルトスリング（合成繊維スリング）の使用荷重を掛け方・角度・安全率で判定。

**根拠・数式**:
- クレーン等安全規則 第218条（繊維ロープ等の不適格品の使用禁止）／玉掛用具の一般則。
- 繊維スリングの安全率は **JIS B 8818（ベルトスリング）** 及び製品の使用荷重（WLL）に従う。
  一般に **使用荷重 = 最大使用荷重（ラベルの色/等級）**、安全率は 6〜7 系（製品規格）。
  → **基本使用荷重（WLL）は製品ラベル/証明書の値を入力**。等級色（JIS の色表示）を select で
  補助。**安全率・WLL の数値は一次資料（JIS B 8818・製造者）で確認**、未確認なら入力方式。
- 掛け方係数（ストレート/バスケット/チョーク）と吊り角度係数（1/cos(θ/2)）を掛ける。
  チョーク（絞り）係数はメーカー公表（一般に0.8前後）。**係数の出典を basis に明記**。

**入力**: `wllKg`（基本使用荷重・製品値）／`hitch`（ストレート/バスケット/チョーク）／
`angle`（°）／`loadKg`（荷質量）。

**外部突合**: メーカー公表の使用荷重表（掛け方・角度別の低減率）と一致。
**強警告**: 角部の当て物（保護材）・鋭角部での切断・紫外線/薬品劣化・ラベル判読不能品の廃棄。

## 1-C. シャックル・アイボルト等 玉掛用具の使用荷重チェック（`rigging-hardware-check`）★P0

**用途**: シャックル・アイボルト・フック等の使用荷重（WLL）と斜め引き低減を確認。

**根拠・数式**:
- クレーン等安全規則 第214条（フックの安全係数）・玉掛用具の一般則。
- シャックル・アイボルトの **使用荷重（WLL）と安全率5系** は **JIS B 2801（シャックル）**・
  製品カタログ・証明書の値。**WLL は入力**（型式・呼びで代表値を select 補助しても良いが、
  数値は製品値優先。勝手な既定値は置かない）。
- **アイボルトの斜め引き低減**（引張角度による許容荷重の急減）は製造者公表の低減表
  （例：45°で約1/3等）。**低減率の出典を明記**、未確認なら入力方式。

**入力**: 器具種別（シャックル/アイボルト/フック）／WLL（製品値）／作用荷重／
（アイボルトのみ）引張角度。

**外部突合**: JIS B 2801 / メーカーの使用荷重・斜め引き低減表と一致。
**強警告**: アイボルトの斜め引き厳禁（原則軸方向）・ピン/ナットの確実な取付・変形品の廃棄。

## 1-D. つり上げ装置（フック・巻上ワイヤ）の定格・安全率確認（`hoist-rated-check`）★P1

**用途**: 巻上ワイヤ・フックの安全係数と、定格荷重に対する余裕を確認（クレーン則の器具安全率）。

**根拠・数式**:
- クレーン等安全規則 第213条（ワイヤロープの安全係数6）・第214条（フックの安全係数4）。
- 巻上ワイヤ：切断荷重 ÷ 6 ≥ 実荷重／フック：破断荷重 ÷ 4 ≥ 実荷重。
  切断荷重・破断荷重は **JIS/製品値を入力**。
- `crane-rated-load`（必要定格総荷重）と相互リンク：本機は**器具側の安全率**、逆引きは**機械側の定格**。

**入力**: 実荷重／巻上ワイヤ切断荷重／フック破断荷重。
**外部突合**: 条文の安全係数（6・4）そのものと、切断荷重÷係数の手計算一致。
**強警告**: 定期自主検査（クレーン則）・端末処理・キンク/素線切れの使用禁止。

## 実装順・PR分割

1-A → 1-B → 1-C → 1-D。各機 1PR。玉掛けワイヤ（`sling-wire-load`）・
クレーン逆引き（`crane-rated-load`）と相互リンクを basis/cautions に必ず入れる。
出典（JIS 番号・条文）は `basis` に版とともに明記。**径別・等級別の切断荷重表は一次資料で
行ズレに注意**。確定できない数値は入力方式にし、REQUIRES-OPUS には回さない（入力方式で解決可能）。

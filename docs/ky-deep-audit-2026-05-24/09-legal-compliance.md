# 09. 法令遵守チェック（軸8）

調査方法: Web調査。条文は e-Gov 原文に直接到達できず（403）、労働新聞社等の転載で複数ソース・クロス確認した。**条番号・見出しは複数ソースで一致を確認済みだが、社外公表前に e-Gov（elaws.e-gov.go.jp）原文での最終照合を推奨**。

---

## 1. 最重要: 前提の訂正

オーナー指示書にあった「**安衛則第28条（危険予知活動との整合）**」は、**条文の取り違え**です。

- **安衛則 第28条 = 「安全装置等の有効保持（点検及び整備）」**であり、危険予知活動の条文ではない（3ソースで確認）。
- 危険予知活動（KY/KYT）を名指しで義務づける単独の安衛則条文は**存在しない**。KY・指差呼称・4ラウンド法は、**中央労働災害防止協会（中災防）のゼロ災運動・KYT** と **建災防** が推進する**自主的安全活動**。
- KY/リスクアセスメントの**実質的な法的アンカーは「安衛法 第28条の2」=リスクアセスメントの努力義務**（平成18年4月施行。義務ではなく努力義務）。「危険性又は有害性等の調査等に関する指針」あり。

**当サイトは既に正しい**: コード内（content-quality-cleanup, law-metadata 等）でKYの根拠を「**安衛法第28条の2 / 第59条の3 / 安衛則第34条の2の8 / 中災防KY手引き**」と正しく引いている。`/ky`画面が安衛則28条を誤って法的根拠に挙げている箇所は確認されなかった。
→ **対応: オーナーの認識を28条の2へアップデートすれば足りる。サイト側の修正は不要。**

## 2. 法令・指針マップ（根拠 → 要求 → 電子KYとの関係）

| 根拠 | 何を求めるか | 電子KYとの関係 |
|------|--------------|----------------|
| **安衛法 第28条の2** | 設備・原材料・**作業行動**等に起因する危険性・有害性の**調査と措置**（努力義務） | RA型KYが支える対象。当社のL×S評価・再評価は要件に合致。記録の保存・更新が電子で可能。 |
| 危険予知活動（KY/KYT） | 法定義務ではない自主活動。4R法・指差呼称（指差呼称はエラー率を大幅低減との実験結果あり） | **様式の法定義務が無い＝電子化の自由度が高い**。ただし4R構成・行動目標・指差呼称を備えてこそ“KYとして成立”。 |
| **安衛法 第29条** | 元方事業者は関係請負人・労働者の法令違反を**指導**、違反者に**指示** | 元請指導の証跡としてKY記録（電子可）が有用。 |
| **安衛法 第30条**（特定元方＝建設業等） | 協議組織、**作業間の連絡調整**、巡視、教育援助、（建設業）作業計画・機械配置計画 等 | KY/TBMが連絡調整・巡視の運用実態を支える。電子で連絡調整ログ化が可能。 |
| **安衛法 第15条の2** | 元方安全衛生管理者の選任（技術的事項の管理） | 元請確認欄の「確認者」に相当しうる。電子承認で代替可。 |
| 作業計画（安衛則155条＝車両系建設機械 等） | 特定危険作業で**作業計画の策定が義務** | KYと別書類だが連動。電子作業計画と電子KYの紐付けが理想。 |
| 監督署（労基署）臨検 | 法定帳簿（労働者名簿・賃金台帳・36協定・健診・委員会議事録 等）を確認。**KY活動表に法定の様式要件・提出義務は確認できず**（任意書類） | KYは形式自由・電子保存可。即時表示/印刷できれば臨検対応に足る。 |
| 電子署名法 | 一定要件の電子署名は**手書き署名・押印と同等の証拠力** | 職長印・作業員サイン・元請確認印は電子署名/電子承認で代替可（本人性・非改ざんが要件）。MHLWは2021年10月以降、届出等の電子署名添付を不要化。 |

## 3. 電子KYが満たすべきチェックリスト（実務）

1. **法的位置づけを正しく表示**: 「KYは自主活動、RAは努力義務（安衛法28条の2）」と明記。**「法定義務」と過剰に書かない**（誇大表示の回避）。
2. **4R構成の保持**: 1R現状把握／2R本質追究（危険のポイント）／3R対策樹立／4R目標設定（重点実施項目・チーム行動目標）を**必須項目化**。現状は4Rの締め（行動目標・指差呼称）が欠落（→02）。
3. **記録の本人性・非改ざん性**: 署名/承認ログ・編集履歴・タイムスタンプ。電子署名法準拠なら押印同等。
4. **元方（元請）確認フロー**: 法29・30・15条の2を踏まえ、元請確認欄＝電子承認を用意（→08）。
5. **関連書類との紐付け**: 作業計画書・作業手順書・施工体制とのリンク（将来構想）。
6. **保存**: KY自体に長期保存義務は無いが、石綿・特化物等の作業記録は最長30年。電子KYが兼ねる場合は保存年限・検索性を担保。現状は端末内30件・自動削除＝記録保全に不安（→06）。
7. **臨検対応**: 即時表示・印刷可能であること（PC/タブレット）。

## 5. 法令観点の判定

- **絶対やる**: ①オーナーの「安衛則28条」認識を「安衛法28条の2＋自主活動」へ更新（社内・営業トーク含む）。②4Rの締め（行動目標・指差呼称）の欄追加。
- **やった方がよい**: 記録保全（クラウド/履歴/保存年限）と元請確認の証跡化。
- **注意（やってはいけない）**: KYやAIに対し「法定義務」「AIが自動判定」等の**過剰・不正確な表現**を付けないこと（擬似AIを“AI”と表記している点は表現の見直し対象）。

# 12 「みんなが専門家レベルで気軽に分析できる」基準達成度（軸12）

## 基準別評価
1. 専門知識ゼロでも分析できるか: 統計ダッシュボード（業種別/月次/比較/予防チェックリスト）は充実。ただし**AIによる「で、何に気をつければいい？」の即答が無い** → 専門家の解釈を補えていない。達成度 ★★★☆☆。
2. グラフ・図表の直感性: monthly-trend/comparison chart あり、3デバイス崩れなし。★★★★☆。
3. 「気軽さ」（操作秒数/クリック数）: /accidents-reports は業種選択から分析に入れるが、初見の「まず何を見るか」導線がやや弱い。スマホでの即答性に伸びしろ。★★★☆☆。
4. 直近性: 統計はmock（2021-2025）、月次速報未取込。「直近がわかる」は未達。★★☆☆☆。

## 総合達成度（推定）: 約70/100
- 強い: 多観点の統計ダッシュボード、業種比較、予防チェックリスト、印刷、3デバイス対応。
- 弱い: AI解釈（専門家レベルの即答）が無い、直近事故の鮮度、KY等現場フローへの差し込み、多言語。

## 「専門家レベルで気軽に」へ詰める順番
1. **AI事故注意喚起**（業種・作業→類似事故＋危険ポイント＋対策。専門家の解釈を無料・即時に）（軸7）。←最重要
2. KY→事故サジェスト動線（現場フローで気軽に）（軸10）。
3. 直近事故の公式リンク＋鮮度表示→将来は月次速報取込（軸6）。
4. 多軸AND絞り込み・関連事故サジェスト強化（軸5）。
5. 多言語（軸8）。

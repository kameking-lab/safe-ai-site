# 08 多言語対応（軸8）

## 現状
- 事故DBは日本語のみ。サイト共通 language-context は ja限定NOOP（化学物質RAの監査で判明）。
- サイネージ（signage-labels）・化学物質RA（chemical-ra-labels）で「局所辞書＋ローカルトグル」の6言語パターンを確立済み。

## 設計案
- M-1 固定ラベル6言語: 事故DBの主要画面ラベル（見出し・絞り込み・ボタン）を chemical-ra-labels 同様の局所辞書で6言語化（日/英/越/中/タガログ/インドネシア）。確定済みUI用語のみ（不確実は日本語＝創作回避）。
- M-2 重要事故ケースの母語要約: AI翻訳併記（将来、軸7のAIと連動、参考明記）。

## 優先度
- P2: M-1 固定ラベル6言語（パターン流用・低リスク）。事故本文は日本語のまま。
- 将来: M-2 ケース母語要約（AI翻訳・コスト/精度検証）。

注: 技能実習生・特定技能者が「自業種で起きやすい事故」を母語で把握できる価値は高いが、事故本文の機械翻訳は精度・責任の観点で「参考」明記必須。

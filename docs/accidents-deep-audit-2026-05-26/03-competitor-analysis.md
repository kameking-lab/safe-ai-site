# 03 競合分析（軸3）

研究エージェント調査 2026-05-26。実在確認できたもののみ。

## 実在確認できた競合
1. **職場のあんぜんサイト（厚労省/JNIOSH）** — 最強競合。労働災害事例 約2,632件＋ヒヤリハット、死亡災害DB(平3〜令5全数)、死傷DB(平18〜令3の1/4抽出)、統計表/Power BI。完全無料・登録不要。絞り込み=業種/事故型/起因物/キーワード。
2. **安全衛生情報センター(JAISH)** — 事例2,500件超、類型別、安衛法令検索。無料・登録不要。
3. **中災防(JISHA)** — 業種別分析データ(PDF)、度数率/強度率。一部統計は会員専用。
4. **建災防** — 建設業特化。災害事例集(PDF)、工種別/災害種別統計。無料。
5. **HACARUS KY** — 商用AI-KY SaaS。約3,500事例DB＋作業条件→AIリスク/KY提案、スマホ。初期10万＋月3万(500シート)。
6. **日立ソリューションズ 労働安全衛生管理** — 商用。災害/ヒヤリ蓄積をKY/TBMで参照。料金不明。

## 職場のあんぜんサイト プロファイル
- 機能: 事例検索(2,632)、死亡災害全数DB、死傷DB、統計表、ヒヤリハット。絞り込み=業種/事故型/起因物/年・月・時間/年齢。Excel DL＋ワードクラウド。
- 料金: 無料・登録不要。
- デバイス: **DL前提・PC寄り**（スマホ最適化の明記なし）。
- 鮮度: 死亡DB令和5年/死傷DB令和3年止まり=**更新ラグ大**。
- 強み: 公的・無料・件数最多・全数死亡DB・二次分析可。
- 弱み: 直近性弱・UI古い・**AI支援なし**・現場運用ツール非連携・クロス分析弱い。

## 当方が刺さる差別化（5件）
1. **スマホ完結の現場即時利用**（公的DBはDL/PC前提）。検索→KY/朝礼へ直結。
2. **AI支援の多観点分析**（公的DBは検索止まり。HACARUSは有料・登録要、当方は無料・無登録）。
3. **直近事故の鮮度**（公的DBは数年ラグ。月次速報取込で優位化余地）。
4. **現場ツール一体運用**（KY/サイネージ/打合せ書/化学物質RAと事故DB連携。公的・商用ともこの統合形なし）。
5. **無料・登録不要のゼロ摩擦**（商用は契約/初期費/会員の壁）。

## 競合にあり当方に無い、有用な機能（5件）
1. 死亡災害の全数データ（あんぜんサイト）。
2. Excel/CSV一括DLでの二次分析。
3. イラスト付き「発生状況・原因・対策」解説（教育素材）。
4. 業種特化の深い事例・統計（建災防/中災防）。
5. 自社事例の追加・蓄積→AI反映（HACARUS）。

## 結論
当方の勝ち筋は **「スマホ完結 × AI多観点分析 × 直近事故 × 現場ツール一体 × 無料登録不要」**。あんぜんサイトの網羅性・全数死亡DBは追わず（取込はリンク誘導＋月次速報）、体験完成度で勝負。

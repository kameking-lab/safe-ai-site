## 裁判例（軸H）の誠実性
- 追加**7件（16→23）**は**すべて実在確認済みの確定判例**。出典＝厚労省「あかるい職場応援団」・裁判所裁判例検索/CiNii・労働新聞社・労働基準判例検索等の信頼できる一次/公的情報。**架空・記憶頼みは1件もなし**（過去に誤って除外した4件と同じ厳格さ）。
- 追加した論点＝社長名指しの 過労自殺(オタフクソース)・パワハラ自殺/いじめ自殺(誠昇会北本共済病院・川崎市水道局)・パワハラ(ザ・ウィンザーホテルズ・日本ファンド)。さらに**論点の異なる「業務起因性＝労災認定」**2件(名古屋南労基署長(中部電力)・国行橋労基署長(テイクロ九州))を#364で追加し、「安全配慮義務(民事)」と「業務起因性(労災行政)」の両系統をカバー。
- **確証取れず見送った論点（正直に）**: 建設墜落・はさまれ・感電・崩壊等の物理災害は、事件名・裁判所・判決年月日を出典で一貫して確証できる「landmark確定判例」が見当たらず、捏造回避のため今回は見送り。確証が取れ次第追加する。

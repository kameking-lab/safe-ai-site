# マスコット スタイルガイド＋必要箇所マップ（2026-07-12）

サイト公式マスコット（ロングコートチワワ）の視覚仕様の正本。
画像生成・加工・配置はすべて本ガイドに従う。キャラの原型は
`web/public/mascot/mascot-chihuahua-4.png`（1024×1024）であり、**この子に見えること**が全生成物の合格条件。

## 1. キャラクター仕様（不変・厳守）

- 犬種: ロングコートチワワ。2頭身デフォルメ。
- 毛色: クリーム（基準色 `#FEF7E7`）。胸元・マズルはより明るく、ふわふわの長毛表現。
- 目: 大きな丸い焦げ茶の瞳（`#2A1C12`）、白ハイライト2点。優しい表情が基本。
- 鼻: 黒く小さい。口元は控えめなにっこり。
- 耳: 大きな立ち耳、縁に長毛のふさ、内側はピンク（`#DF907A`）。
- ヘルメット: 緑（`#159968`）の工事用ヘルメット、正面に**白い十字（緑十字）**。
  緑十字は安全衛生の象徴であり、このキャラの身元そのもの。**色変更・省略禁止**。
- 線: 太い濃緑のアウトライン（`#0A1C10`）、ステッカー風の均一な輪郭。
- 塗り: フラットなセル塗り。グラデーション・写実的陰影は使わない。

### カラーパレット（実測値）

- 緑（ヘルメット/背景円）: `#159968`
- クリーム（毛色）: `#FEF7E7`
- ピンク（内耳）: `#DF907A`
- 焦げ茶（瞳）: `#2A1C12`
- 濃緑黒（アウトライン）: `#0A1C10`
- 白（十字/リング/ハイライト）: `#FFFFFF`

### JIS安全色との整合（不可侵）

- 緑 `#159968` はJIS安全色の「緑=安全状態・衛生」の意味論と整合。緑十字ヘルメットは安全衛生旗と同じ記号体系。
- マスコットの装飾に**赤（禁止）・黄（警告）・青（指示）を意味なく使わない**。
  警告的な文脈（disallowed等）にマスコットの緑を流用しない。
- 高視認ベスト等の小物を足す場合も、黄/オレンジの面積を大きくしない（JIS意味論との喧嘩を避けるため、小物も緑系・白・クリームで統一）。

## 4. 品質判定（キュレーション基準）

生成物は必ず画像として開いて目視し、以下で採否を決める。理由はメタデータ（output/<id>/meta.json のcuration欄）に記録する。

1. 同一性: 参照画像と同じ個体に見えるか（毛色・目・耳・ヘルメットの緑十字）。
2. スタイル適合: 太アウトライン・フラット塗り・2頭身が守られているか。
3. 記号の正確性: 十字が緑地に白十字か（赤十字は不可=赤十字社標章と衝突）。
4. 透過品質: 背景が真に透過か（白残り・フリンジは後工程で除去可能な程度か）。
5. 用途適合: 置き先のサイズで潰れない構図か。

## 5. 重量・実装予算（不可侵）

- 配置画像は webp（アルファ付）、表示サイズの2倍以下にリサイズ。
- 1枚あたり重量予算: 装飾スポット ≤30KB、ヒーロー ≤60KB。
- 全imgに width/height 明示（CLS 0維持）、装飾は `loading="lazy"`（ファーストビューのヒーローのみeager可）。
- Lighthouse Performance/Accessibility 90+ を維持。before/after記録必須。

# 05 公開と権利 — 利用規約案・配布形式

- 競合実査(07)の結論を反映: 「申請不要・条件明文・編集可」それ自体が差別化。CC系ライセンスは社内利用=商用の解釈が割れる(NC条項の曖昧さ)ため採用せず、**独自の短い利用条件**を教材・ページ両方に明記する。
- 文言の最終確定は労働安全コンサルタントであるオーナーの専権(要オーナー確認)。以下は起案。

## 1. 利用規約(案・全文骨子)

「ご自由にお使いください」の範囲を6条で明文化する。

1. **できること(無償・申請不要)**: 事業場内の安全衛生教育(特別教育の学科・雇入れ時教育・TBM/朝礼・社内勉強会等)での投影・印刷・配布。自社向けの改変(社名・現場写真・自社事例の追加、順序変更)。企業・団体・登録教習機関を含む、営利組織内での利用。
2. **出典明記**: 表紙または最終ページの「提供: 安全AIポータル(労働安全コンサルタント監修)+URL」クレジットを残すこと。改変版でも「安全AIポータルの教材を基に作成」の一文を残す。
3. **できないこと**: 教材そのもの(改変版を含む)の販売・有償配布。自作教材としての再公開(クレジット除去)。当サイトの名や「法定充足保証」表示を、改変後の教材に残すこと(下記4)。
4. **保証の失効条件(本パックの核)**: 「法定科目・時間の対応表」と充足保証は**無改変の原本にのみ**適用される。スライドの削除・科目に関わる編集を行った場合、対応表・保証バッジは失効する(改変時は対応表ページを削除するか「改変済み・充足未検証」と付記すること)。改変を禁止しないが、保証と改変は両立させない。
5. **教育実施の責任**: 本教材の利用は労働安全衛生法上の教育の「実施」ではない。科目・時間・講師・記録等の法定要件の充足は実施する事業者の責任(01の線引き文言と同一。教材にも同文を焼き込む)。
6. **免責と更新**: 内容は作成時点の法令・統計に基づく(各教材にデータ基準日・準拠改正を印字)。法改正・統計更新により内容は予告なく更新される。最新版は常に本サイトのURLで提供。

再配布の扱い(起案): 原本無改変+無償+クレジット維持ならOK(社内サーバー・元請から協力会社への横流しを止めない — 現場実態に合わせる)。ただし「最新版は本サイトに」の一文で更新遅れリスクをヘッジ。

## 2. 配布形式

- **Web投影(第一形式)**: 既存 SlideDeck の present モード(16:9・キーボード/タップ送り・URLハッシュ共有・WakeLock)。常に最新データ。ネット環境のある教育はこれが最良。
- **印刷/PDF**: SlideDeck の print モード(A4横・改ページ固定)+ブラウザのPDF保存。配布資料・記録添付用。
- **PPTX(編集可・目玉)**: 既存セミナーPPTXパイプライン(`data/seminars/*.yaml` → `scripts/seminar_pptx.py`(python-pptx) → `web/public/seminars/*.pptx`)を流用。**新規依存パッケージ不要**(オーナー確認事項から除外できる)。現行の10枚サンプルを、02のゲートを通したフルデッキ(科目網羅+対応表+統計+確認テスト+CTA)へ拡張する。
- 同梱書類: 法定対応表(1枚)・教育実施記録簿/受講者名簿の印刷様式(修了証は同梱しない — 07§3)・利用条件(1枚)。

## 3. ページ上の見せ方

- 各教育詳細ページのダウンロード区画を「サンプル」から「フル教材(無償)」へ改称し、利用条件の要約3行(出典明記/改変可/販売不可)+全文リンクを添える。ダウンロードに申請・登録・パスワードは置かない(競合の摩擦がそのまま当方の優位)。
- 「ご自由にお使いください」の直下に、保証失効条件(§1-4)を1行で: 「改変された教材には法定対応表の保証は適用されません」。
- 利用規約全文は `/education` 配下の独立ページ(または既存 /terms への追記でなく教材専用ページ)として公開し、全教材・全PPTXから同一URLで参照。

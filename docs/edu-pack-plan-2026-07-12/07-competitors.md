# 07 競合調査 — 無料教材の実態と本パックの空白地帯(2026-07-12 Web実査)

調査方法: WebSearch+一次ページ取得(WebFetch/curl)。取得不能だった項目は末尾「要オーナー確認リスト」に正直に残す(憶測で埋めない)。

## 1. 競合マップ(4象限の結論)

- **公共(厚労省・職場のあんぜんサイト)**: 全て無料だが PDF・断片動画中心。フルハーネス動画(YouTube)・未熟練労働者教育マニュアル(一部PPTX)・熱中症ガイド(PDF+MP4、「雇入れ時教育に活用ください」明示)はあるが、**特別教育の学科をフルカバーする無料スライド一式は存在しない**。法定科目・時間の充足保証もほぼ無し。ライセンスは公共データ利用規約(PDL1.0)=出典明記で複製・改変・商用可 → **本パックの素材源として最重要**。
- **業界団体(建災防・中災防)**: 教育の中核教材は有償テキスト(建災防フルハーネステキスト1,045円・令和7年4月第5版等)。中災防の講義用PowerPoint(講義用図表集)は**テキスト20冊以上購入の特典**でパスワード制。無料領域はリーフレット・啓発動画のみ。販売物は「画像の加工・二次使用・営利目的の使用・データの再配布はご遠慮ください」と改変・再配布禁止。
- **民間無料勢(中小建設業特別教育協会・安全衛生マネジメント協会)**: 法定カリキュラム構成に沿うWEB教材・熱中症勉強会スライド(PDF)を持つが**全て「要:使用申請」**で、編集可能なPPTXではなく、改変可否・商用利用条件が不明瞭。
- **フルパック(valley-up)**: 科目別PPTX一式+記録簿・名簿・修了証テンプレ+科目別テストという「そのまま特別教育を実施できる」構成が存在するが、**同社の有償講師養成講座の修了者限定・開封パスワード制**。
- **メーカー(3M・谷沢・藤井電工・ミドリ安全)**: カタログ・取説・製品教育動画のみ。法定充足は謳わず、教育スライドの配布実態なし。
- **個人コンサル(柳川労働安全衛生コンサルタント事務所)**: 熱中症予防管理者講習用の生PPTを公開・自社向け改変可・2025年改正対応明示。ただし**商用(有償研修)利用禁止**・個人サイトで組織的担保なし。最も思想が近い先行例。

## 2. 本パックが取る空白(=線引きの具体化)

1. **「申請不要・PPTX/PDF編集可・法定科目/時間対応表つき」のフル教材は市場に存在しない**。4象限(公共=断片/団体=有償/無料勢=要申請/フルパック=限定)の交点が空白のまま。
2. **法定充足の「保証」を機械化しているものは皆無**。競合の「準拠」は全て自己申告の静的文言。本パックは e-Gov 正本スナップショット×CI照合(02)で、対応表と検証日を教材自体に印字する — 労安コンサルの看板と機械保証の二重担保は他に無い。
3. **統計の鮮度で全競合が止まっている**(厚労省eラーニングは令和2年度版、JOHASの熱中症PPTは2009年、団体テキストは年〜数年改訂)。月次ETL連動+データ基準日印字(04)は構造的優位。
4. **利用条件の明文化それ自体が差別化**。競合はほぼ「要申請」「条件不明」「商用不可」。「出典表示のみで社内教育・改変自由」を明文で出す(05)。
5. **熱中症は需要最盛期に供給が最弱**: 2025年6月義務化(安衛則改正)+クールワークキャンペーン継続に対し、教育スライド一式は厚労省PDFと個人サイト(商用不可)のみ。初弾に最適(03)。

## 3. 競合から学ぶ構成要素(パクるべき形)

- valley-up 型の「運用書類同梱」: 教育実施記録簿・受講者名簿の印刷様式を同梱する(安衛則38条の記録3年保存に対応)。**ただし修了証テンプレは同梱しない** — 「修了証発行に参入しない」戦略(docs/nihonichi-strategy-2026-07-12.md の線引き)を維持し、NIQ-REC2(社内記録用の受講記録エクスポート・法的効力なしの注記固定)と同じ免責文言を様式に焼き込む。
- 中災防の「講義用図表集」概念: 講師が使う前提の構成(スライドノートに講師向け補足)は本パックでも踏襲する価値あり。
- 柳川事務所の「改正対応の明示」: 準拠改正・ガイドライン版数を表紙に書く作法。

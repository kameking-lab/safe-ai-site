# visual-assets-wanted — AI生成画像の必要リスト（Path A・オーナー生成）

柱0（脱テキスト・ビジュアルファースト）の画像分業リスト。
**運用**: ループ側がここに「必要画像＋生成プロンプト案＋配置先」を書き出す → 社長側でGemini等で生成し
`web/public/illustrations/` に指定ファイル名で配置 → ループが次イテレーションで組み込む。

- 第一選択は常にインラインSVG・lucide・CSS自作（ライセンス問題ゼロ）。ここに載せるのは「真に絵が効く箇所」のみ。
- 生成画像は **商用利用可の生成サービス** で作成し、生成日・サービス名をこのファイルに追記して記録する（ライセンス記録）。
- 画像内に文字を入れない（多言語ページで言語が固定されるため。文字はHTML側で重ねる）。
- 推奨フォーマット: WebP（写真調）/ PNG透過（挿絵調）、横長は1200×630目安、挿絵は800×600目安。

## ステータス凡例
- `WANTED` = 生成待ち / `PLACED` = public配置済み・組み込み待ち / `DONE` = 組み込み済み

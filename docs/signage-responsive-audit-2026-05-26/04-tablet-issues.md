# 04. タブレット 縦・横（768×1024〜1024×1366）

シナリオ: 朝礼での共有閲覧、iPad片手持ち→机置き、立てかけ表示。

## 実測
- iPad 768×1024（縦）: 実1784px → **+760px 超過**
- iPad Pro 12.9" 1024×1366（縦）: 実1871px → **+505px 超過**
- 横向き（1024×768 等）: 高さ768pxに対し縦積みで大幅超過（PCノート同様、doc05）

## 所見
- 縦向きでも全項目は1画面に収まらず、共有閲覧で下部が見えない。
- 横向きは高さがさらに低くなり超過が拡大。

## 改修方針（→doc11）
- タブレットは**表示系**として1画面フィット適用。
- 横向き・幅広時は**2カラム化**で高さを圧縮。縦向きは余白圧縮＋自動縮尺で収める。
- タッチ操作（言語/全画面切替）の標的44px以上。

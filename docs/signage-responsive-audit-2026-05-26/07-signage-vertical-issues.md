# 07. デジタルサイネージ 縦置き（1080×1920）

シナリオ: 縦長ディスプレイ（現場入口・通路の縦型モニター）。

## 実測
- 1080×1920: 実1920px → **超過なし（唯一フィット）**

## 所見
- **縦型は単一カラム縦積みと相性が良く、現状で唯一1画面に収まる**。設計の縦方向の流れ自体は正しい。
- ただしフィット倍率最適化（高さに合わせて文字を最大化）や上下の余白バランスは改善余地。動的コンテンツ（安全啓発の長文・6言語）次第で僅かに溢れる可能性。

## 改修方針（→doc11）
- 縦型は**現状の単一カラムを維持**しつつ、1画面フィット（高さ連動の縮尺）を適用して常に収まりを保証。
- 縦型では2カラム化しない（縦長に不適）。orientation/aspect比で分岐。

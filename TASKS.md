# TASKS.md

## ルール
- 上から順に1つずつ実行する
- 1タスク完了ごとに progress.md を更新する
- 途中でエラーが出たら自分で修正して続行する
- 判断に迷ったら PRD.md を優先する
- MVP外の機能は実装しない

## タスク一覧

2. TypeScript を有効にする
3. Tailwind CSS を導入する

5. 開発サーバーが起動することを確認する

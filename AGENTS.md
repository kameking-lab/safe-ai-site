# AGENTS.md

## このプロジェクトのゴール
安全AIサイトを「本当に使える現場運用ポータル」として完成させる。
朝礼・常時表示・個人確認の3用途で日常的に使われ、安全グッズのアフィリエイトで収益化も行う。

## 機能一覧
- 法改正一覧（直近10年分、キーワード・年フィルタ、AI要約、質問チャット）
- 今日の現場リスク（地域選択・作業種別・朝礼ブリーフィング）
- 事故データベース（既存事故DB + 厚労省 死亡災害/休業災害DB 10年分モック、検索可能）
- /signage サイネージ画面（日本地図ハッチング、報道労災、法改正要約）
- Eラーニング（テーマ一覧・クイズ・学習状態）
- KY用紙（紙様式準拠の危険予知活動表）
- PDF出力（KY・朝礼要点のプレビュー＋印刷）
- 通知/メール配信設定
- 音声入力（全テキスト入力欄対応）
- 安全グッズ（分野別カテゴリ、アフィリエイトリンクで収益化）

## 最重要ルール
- エラーが出たら自分で修正して続行する
- ユーザー確認なしで進める
- 動作確認を優先する
- 実装後は変更ファイルを簡潔にまとめる
- 変更のたびにオンラインに反映（Vercel デプロイ）する

## 技術ルール
- Next.js App Router を使う
- TypeScript を使う
- Tailwind CSS を使う
- コンポーネントを分ける
- APIキーや秘密情報は絶対に埋め込まない

## UI方針
- スマホでは縦長スクロールで見やすく
- PCでは横長画面を最大限活用（サイドバー + メインコンテンツが広く使える）
- サイネージはフルスクリーンで朝礼向け

## 禁止事項
- 外部APIの本接続（**例外**: サーバー側 Route Handler からの **Open-Meteo** 等・**APIキー不要**の公開気象APIのみ可。キー付き外部サービスや任意の有料APIは不可）
- DB本接続
- APIキーや秘密情報の埋め込み

## 許可する範囲
- ページの自由な追加・改造
- 音声入力のクライアント側実装（ブラウザAPI）
- PDF出力（プレビュー、印刷、将来差し替え可能な構造）
- アフィリエイトリンク（Amazon/楽天等の商品リンク）の設置
- Vercel へのデプロイ

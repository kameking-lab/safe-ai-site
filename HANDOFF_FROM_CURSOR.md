# Cursor → Claude Code 引き継ぎ

## このサイトは何か
労働安全コンサルタント（合格済み）が、協会でシステム受注するためのポートフォリオとして作っている
「労働安全に関する便利サイト（現場運用ポータル）」。
会社の安全担当者が業務時間内に日常的に使う想定。

## 本番URL
- メイン: https://safe-ai-site.vercel.app/
- サイネージ: https://safe-ai-site.vercel.app/signage
- GitHub: https://github.com/kameking-lab/safe-ai-site
- デプロイ: Vercel（GitHub連携で自動デプロイ）

## 技術スタック
- Next.js App Router + TypeScript + Tailwind CSS
- モックデータ中心（本番API/DB未接続）
- 気象庁JSON・Open-Meteo・GoogleニュースRSSは一部ライブ接続済み
- Vercelでホスティング

## Cursorで詰まっていたこと
1. **自律的に動かない**: 指示しないと進まず、自分で課題を見つけて改善しない
2. **データ収集が弱い**: 「10年分集めて」と言っても100件程度で止まる
3. **稼働が止まる**: 長時間の自律作業ができない
4. **レイアウト問題**: PCで見ると全体的に横に伸びて間抜け
5. **レビューが大変**: 都度細かく指示しないといけない

## オーナーが望む運用
- 完成形のイメージだけ示して、Claude が道筋を立てて自律的に進める
- 外出先からスマホでレビューし、修正指示を出す
- 最終判断（公開/マージ）だけオーナーがする
- 課題を自分で発見して提案・修正してほしい

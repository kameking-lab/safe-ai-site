# ANZEN AI 改善タスクリスト

## ルール
- 上から順番に実行する
- 1タスク完了ごとにコミット（英語メッセージ）
- 使用量の10%を超えたと感じたら中間報告して停止
- 数時間に1回は進捗報告
- 完了したタスクは [x] にする
- モックデータ・テストデータは一切使わない。実データのみ。

- [x] 1-2. 事故DB表示順: 最新日付順にソート（2007年が先頭に来ないように）
- [x] 1-3. 法改正の最新5件を確認・修正: 2025-2026年の直近改正が上に来るように
- [x] 1-4. クママップ富山県完成: 「クマっぷ」から直近1年分を500件以上取得。ピンを大きく、種別で色分け（目撃=緑大、被害=赤大、捕獲=青、痕跡=灰）。市区町村テキスト検索欄追加。都道府県選択で地図ズーム連動。
